# Introduction

Common Engineering Policy (CEP) are requirements that all applications must adhere to. Some CEPs apply only to serverless applications, some CEPs apply to microservices, and some CEPs apply to all classes of applications. CEPs have been established to ensure security, quality, and consistency across the Platform+ Tech platform. Some applications may be exempt from implementing a CEP if there are natural constraints (example developing applications using COTS solutions).
