Aside from exploratory and automation testing support, Platform Validation provides accessibility testing support for product teams to ensure our products captures the needs of all users. We are focused on ensuring our products align to the following principles of accessibility: Perceivable, Operable, Understandable and Robust. 
* Accessibility test cases are created for each product against the WCAG 2.1 AA standards, which is the guidelines Platform+ Tech is adhering to
* Accessibility testing is provided via verification from tools such as WAVE Evaluation tool, Narrator or NVDA to determine if any accessibility errors might have occurred 

One of the tooling we use is a browser plugin called Wave Evaluation tool. It can be found on the Chrome web store. It provides the ability to give us quick feedback on potential accessibility errors on the current web view.

After an initial scan, the tool will provide a summary of errors on the left of the browser.

It also highlights various elements on the page which has an error.
