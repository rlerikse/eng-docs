Primarily, Platform Validation team supports the testing of web, mobile, and API products. 

## Web (all desktop and mobile browsers)
- Exploratory Testing 
- Automated Testing (Cypress, Playwright)
- Accessibility Testing 

## Mobile (iOS, Android)
- Exploratory Testing 
- Automated Testing (Appium, XCUITest) 
- Accessibility Testing 

## API 
- Exploratory Testing 
- Automated Testing (Postman)
